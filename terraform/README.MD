# Terraform


## comandos

* Inicia um repositoria terraform
    terraform init

* Test de criação
    terraform plan

* Cria o terraform
    terraform apply

* Deletar o terraform
    terraform destroy

* Valida o funcionamento do arquivo
    terraform validate

* Indica melorias de formatação
    terraform fmt -check=true -write=false -diff=true
