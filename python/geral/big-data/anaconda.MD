# Anaconda

- Ativar
```
conda activate
```

- Desativar
```
conda deactivate
```

- Python suportado
```
conda search python
```

- Criar ambiente
```
conda create -n NAME_ENV python=2.7 anaconda
```

- Deletar ambiente
```
conda remove -n NAME_ENV --all
```

- Mostra ambienetes ambientes criados
```
conda info --envs
```

- Mudar de ambiente
```
conda activate NAME_ENV
```

- Instalar pacote
```
conda install -n NAME_ENV pacote
```

- Remove pacote
```
conda remove -n NAME_ENV pacote
```
